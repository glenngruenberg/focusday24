# Backend 

To be compatible with the frontend we require the following endpoints and returns

| Type | Route |
|------|-------|
|POST  | /sign-in |
|GET   | /sign-out |
|GET   | /times/:date |
|GET   | /times/:id | 
|POST  | /times |
|PATCH | /times/:id |
|DELETE| /times/:id |
|POST  | /vtt |
|GET   | /tickets/:userid?status&startdate&enddate |
